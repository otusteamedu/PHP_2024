Алгоритмическая сложность определения цикличности в списке - O(n), т.к. задача решается за один проход по связанному списку.

Максимальная алгоритмическая сложность нахождения комбинации букв O(n * 4^n): Сначала идет цикл по цифрам, внутри цикла есть цикл перебора по буквам для каждой цифры. Максимальнок кол-во букв для цифры - 4.
