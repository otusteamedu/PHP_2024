Сборка образа
--------------------
```shell
make build
```

Запуск контейнера (сервер)
--------------------
```shell
make start-server
```

Запуск контейнера (клиент)
--------------------
```shell
make start-client
```

Остановка контейнера (сервер)
--------------------
```shell
make force-stop-server
```

Остановка контейнера (Клиент)
--------------------
```shell
make force-stop-client
```
