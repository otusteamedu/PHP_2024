# Docker Compose

Установите Docker

## Установка и запуск

Соберите контейнеры Docker с помощью Docker Compose:

```bash
    docker-compose up -d
```

Откройте файл hosts, добавив следующую строку в конец файла:

```
    127.0.0.1 application.local
```

## Доступ к приложению

URL: http://application.local
